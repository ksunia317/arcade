# Техническое задание
Игра в жанре roguelike, игрок может начать забег, после попадает в лабиринт, который динамически обновляется с течением времени, после определенных действий попадает на новый уровень лабиринта, после выполнений действий на последним уровне он побеждает. в случае смерти, начинает заново. Способности у героя обновляются на каждом уровне
