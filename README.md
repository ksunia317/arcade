# Техническое задание

## Название проекта
CardMan Arcade

## Жанр и концепция
Roguelike-лабиринт с динамической генерацией уровней. Игрок управляет персонажем, который проходит через серию изменяющихся лабиринтов, получая новые способности на каждом уровне. Смерть приводит к полному перезапуску игры.

## Основные механики
1. **Генерация лабиринтов**:
   - Каждый уровень представляет собой уникальный лабиринт, сгенерированный по алгоритму.
   - Динамическое изменение структуры лабиринта в течение игры (обновление стен, переходов, ловушек).

2. **Система уровней**:
   - Игрок начинает с первого уровня.
   - Для перехода на следующий уровень необходимо пройти лабиринт. 
   - После завершения последнего уровня игрок побеждает.

3. **Способности**:
   - На каждом новом уровне персонаж получает случайную или выбираемую игроком способность.
   - Способности влияют на перемещение, бой, исследование и выживание.

4. **Поражение**:
   - При поражении игрок начинает новый забег с первого уровня без сохранения прогресса.

## Геймплей
- Управление: клавиши WASD для перемещения.
- Интерфейс: отображение здоровья, уровня, текущих способностей.
- Игровой цикл: меню → старт → исследование лабиринта → выполнение цели уровня → переход на следующий уровень → победа или поражение(смерть).

## Целевая платформа
- Desktop (Windows, macOS, Linux) с использованием фреймворка Arcade (Python).

## Технические особенности
- Система коллизий для стен и препятствий.
- Модульная система способностей и врагов.

- Язык: Python 3.10+
- Графический фреймворк: Arcade
