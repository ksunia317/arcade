# Техническое задание: Игра "GAME"

## Общее описание

Игра представляет собой карточную текстовую игру с визуальным интерфейсом, реализованную на Python с использованием
библиотеки Arcade. Игрок принимает решения, выбирая варианты ответов на карточках-ситуациях. В игру интегрированы три
мини-игры: Flappy Bird, Platform Jumper и Arkanoid.

### Основной геймплей (Run)

**Система комнат (rooms)**:

- Каждая комната содержит карточку с ситуацией и фоновым цветом.
- Игрок последовательно проходит комнаты.

**Интерфейс**:

- Верхняя панель (UI) с разделителями
- Отображение текущей карточки
- Визуальное выделение выбранного варианта

**Управление**:

- Выбор варианта мышкой (клик по кнопкам)
- Обработка кликов в области 350x200 и 450x200

**Логика карточек**:

- Каждая карточка содержит:
    - Текст ситуации
    - Текст для кнопок
    - Функции-обработчики для согласия/отказа
    - Визуальные параметры (цвета, текстура, масштаб)
- При выборе варианта вызывается соответствующая функция, изменяющая состояние игрока

### Мини-игры

- Каждая мини-игра реализуется как отдельный arcade.View
- Возможность перехода из основного геймплея
- Возврат в основной геймплей с результатами

1. **Flappy Bird**:
    - Управление птицей для пролета через препятствия
    - Очки за пройденные трубы
    - Учет рекорда

2. **Platform Jumper**:
    - Платформер с прыжками по платформам
    - Сбор предметов
    - Уровни сложности

3. **Arkanoid**:
    - Классический арканоид с битой и шаром
    - Разрушение блоков
    - Уровни с разным расположением блоков

### Используемые технологии

- **Язык**: Python 3.8+
- **Библиотеки**:
    - Arcade (графика, интерфейс, управление)
    - Встроенные библиотеки Python (sys, random и др.)
