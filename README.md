# Техническое задание

## Название проекта
CardMan Arcade

## Жанр и концепция
Roguelike игрок получает карты, и используя их проходит дальше, новые способности получает на каждом уровне. Смерть приводит к полному перезапуску игры.

## Основные механики
1. **Генерация уровней**:
   - каждый уровень состоит из карт(выборов игрока)
   - игрок может добавлять свои карты, которые находятся в инвентаре, чтобы поменять ситуацию

2. **Система уровней**:
   - Игрок начинает с первого уровня.
   - Для перехода на следующий уровень необходимо пройти лабиринт. 
   - После завершения последнего уровня игрок побеждает.

3. **Способности**:
   - На каждом новом уровне персонаж получает случайную или выбираемую игроком карту.
   - Каждую карту можно применить в определенных ситуациях, чтобы победить, спастись и тд.

4. **Поражение**:
   - При поражении игрок начинает новый забег с первого уровня без сохранения прогресса.

## Геймплей
- Управление: выбор карт на экране мышкой, или же клавиатурой.
- Интерфейс: отображение здоровья, уровня, текущих карт.
- Игровой цикл: меню → старт → исследование лабиринта → выполнение цели уровня → переход на следующий уровень → победа или поражение(смерть).
