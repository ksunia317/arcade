# Пояснительная записка к проекту «CardMan Arcade»


Проект представляет собой игровое приложение "CardMan", написанное на Python с использованием библиотеки Arcade. Приложение включает главное меню, основную игру и несколько мини-игр с классическим геймплеем.

## Архитектура проекта

### Основные модули:

2. **menu.py** - Реализация главного меню
3. **miniGames.py** - Меню выбора мини-игр
4. **miniGameFlappyBird.py** - Мини-игра Flappy Bird
5. **miniGamePlatformJumper.py** - Мини-игра Platform Jumper
6. **miniGameArkanoid.py** - Мини-игра Arkanoid
6. **run.py** - игра КАРТЕЖНИК

## Реализованные функции

### 1. Главное меню (MenuView)
  - Заголовок "CardMan" 
  - Кнопки: "Играть", "Настройки", "Мини-игры"

### 2. Меню мини-игр (MiniGamesView)
- **Кнопки выбора игр**:
  - Flappy Bird
  - Platform Jumper
  - Arkanoid
  - КАРТЕЖНИК
- **Кнопка выхода**: В главное меню (с текстурой кнопки)

### 3. Мини-игры

#### 3.1 Flappy Bird
**Механика игры:**
- Управление птичкой пробелом
- Преодоление препятствий (трубы)
- Система подсчёта очков

**Функциональность:**
- Анимация крыльев птицы
- Генерация случайных препятствий
- Обнаружение столкновений
- Пауза (Shift)
- Экран Game Over
- Перезапуск игры (Пробел)

#### 3.2 Platform Jumper
**Механика игры:**
- Платформер с вертикальной прокруткой
- Управление стрелками (влево/вправо) и прыжок (вверх)
- Динамическая камера, следующая за игроком
- Генерация случайных платформ

**Функциональность:**
- Физический движок Arcade
- Система подсчёта высоты
- Пауза и Game Over
- Плавное движение камеры

#### 3.3 Arkanoid
**Механика игры:**
- Классический арканоид
- Управление платформой стрелками
- Разрушение кирпичей мячом
- Система жизней (3 жизни)

**Функциональность:**
- Генерация сетки кирпичей
- Физика отскока мяча
- Звуковые эффекты при столкновениях
- Экран победы (при разрушении всех кирпичей)
- Экран Game Over (при потере всех жизней)

## Особенности интерфейса мини-игр

### Универсальные элементы:
1. **Система паузы**:
   - Активация: Shift
   - Панель с кнопками продолжения и выхода в меню
   - Единообразный дизайн во всех играх

2. **Экраны завершения**:
   - Game Over для поражения
   - Экран победы (Arkanoid)
   - Кнопки перезапуска (Пробел) и выхода (ESC)

3. **Навигация**:
   - ESC: возврат в меню мини-игр
   - Согласованная навигация между всеми экранами

### Визуальные элементы:
- Кастомизированные кнопки
- Текстуры фона для каждой игры
- Информационные панели (счёт, высота, жизни)
- Анимации спрайтов

### Технические особенности:
- Использование UIManager Arcade для GUI
- Система столкновений Arcade
- Поддержка разных состояний игры (playing/paused/game_over)
