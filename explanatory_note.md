# Пояснительная записка к проекту «CardMan Arcade»

Проект «CardMan Game Arcade» представляет собой компьютерную игру в жанре roguelike, разрабатываемую на языке Python с использованием графического фреймворка Arcade. Игра сочетает элементы динамически генерируемых лабиринтов, прогрессивной системы способностей и перманентной смерти


## Структура проекта

### Архитектура:
- **Модуль `arcade`**: основной графический и игровой движок
- **Класс `MenuView`**: управление главным меню и навигацией
- **Класс `GameView`**: основной игровой процесс, управление уровнями
- **Класс `Player`**: логика персонажа
